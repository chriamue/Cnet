# Cnet

Unet like network

## Install

```bash
git clone https://github.com/chriamue/Cnet
cd Cnet
virtualenv -p python3 venv
source venv/bin/activate
pip install -r requirements.txt
```

## Train

```bash
python train.py configs/config.yml
```
